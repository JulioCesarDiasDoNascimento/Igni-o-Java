# Anotações

## Utilizando metodos e interfaces de list 

* sort
* Comparator = interface usada para comparar dados de uma list
* Comparator.comparingInt = retorna um comparator que compara valores inteiros extraidos de um objeto
* ToIntFunction = interface usada para converter dados de uma list
* (Problema ter que criar uma classe para isso)

### Solução
#### Expressões Lambda

> Operador de flexa (->)
* isso se chama programação funcional

> Quando tem apenas 1 parametro, pode-se omitir os parênteses
> E também pode ser omitido o retorno
> E também pode usar o method reference (Referencia::Nome do Método)
* Isso vai ser convertido para um lambda.
