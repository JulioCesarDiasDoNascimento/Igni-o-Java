# Anotações 

Optional em Java,nunca retorna null

* Optional< OQUE ELE RECEBE > logica normal
* Se for null ele retorna Optional.empty()

* get() - procura o valor dentro do Optional
* isPresent() - valida se tem ou não valor (boolean)
* ifPresent() - se tiver valor ele faz uma operação
* orElseThrow() - se não tiver valor ele lança uma exceção personalizada
* orElse() - se não tiver valor ele retorna um valor padrão
